# Bugs Arena

An exciting real-time multiplayer arena game where players control bugs in competitive battles. Built with vanilla JavaScript, HTML5 Canvas, and WebSockets for seamless multiplayer experience. Features include real-time collision detection, power-ups, and dynamic gameplay mechanics.

## Features

- **Real-time Multiplayer**: WebSocket-based multiplayer functionality
- **Canvas Rendering**: HTML5 Canvas for smooth graphics and animations
- **Collision Detection**: Real-time collision detection and physics
- **Power-ups**: Dynamic power-up system for enhanced gameplay
- **Responsive Design**: Works across different screen sizes

## Technical Implementation

- **Vanilla JavaScript**: Pure JavaScript implementation for optimal performance
- **HTML5 Canvas**: High-performance graphics rendering
- **WebSockets**: Real-time communication between players
- **CSS3**: Modern styling and animations
