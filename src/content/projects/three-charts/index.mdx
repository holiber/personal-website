# Three Charts

A lightweight JavaScript library for creating interactive charts and data visualizations. Supports line charts, bar charts, and pie charts with customizable styling, animations, and responsive design. Perfect for dashboards and data-driven applications.

## Features

- **Multiple Chart Types**: Line, bar, and pie chart support
- **Interactive Elements**: Hover effects and interactive data points
- **Customizable Styling**: Flexible theming and color options
- **Responsive Design**: Adapts to different screen sizes
- **Smooth Animations**: Fluid transitions and animations

## Chart Types

- **Line Charts**: Perfect for time series data and trends
- **Bar Charts**: Ideal for comparing categorical data
- **Pie Charts**: Great for showing proportional data

## Technical Features

- **Lightweight**: Minimal bundle size for optimal performance
- **Modern JavaScript**: Built with ES6+ features
- **Canvas Rendering**: High-performance graphics rendering
- **Modular Design**: Easy to extend and customize
