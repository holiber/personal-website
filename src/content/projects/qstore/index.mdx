# QStore

A custom query language and database system inspired by MongoDB, designed for working with JSON collections. Features include complex query operations, indexing, filtering, and aggregation. Built as a diploma project to demonstrate understanding of database design principles.

## Features

- **JSON Query Language**: Intuitive query syntax for JSON collections
- **Complex Operations**: Support for advanced query operations
- **Indexing**: Efficient data indexing for improved performance
- **Filtering**: Advanced filtering capabilities
- **Aggregation**: Data aggregation and grouping functions

## Query Capabilities

- **CRUD Operations**: Create, Read, Update, Delete operations
- **Advanced Filtering**: Complex filtering with multiple conditions
- **Sorting**: Multi-field sorting capabilities
- **Pagination**: Efficient data pagination
- **Aggregation**: Data grouping and aggregation functions

## Technical Implementation

- **JavaScript**: Pure JavaScript implementation
- **JSON Processing**: Efficient JSON data manipulation
- **Query Parser**: Custom query language parser
- **Performance Optimization**: Optimized for large datasets
