# QStore

A custom query language and database system inspired by MongoDB, designed for working with JSON collections.
Features include complex filtering, sorting, and aggregation. 
Built as a diploma project to demonstrate understanding of database design principles.


## Capabilities

- **CRUD Operations**: Create, Read, Update, Delete operations
- **Advanced Filtering**: Complex filtering with multiple conditions
- **Sorting**: Multi-field sorting capabilities
- **Pagination**: Efficient data pagination
- **Aggregation**: Data grouping and aggregation functions
