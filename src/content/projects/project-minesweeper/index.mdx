# Minesweeper Game

A faithful recreation of the classic Windows 98 Minesweeper game built with Phaser 3 game engine. Features authentic retro styling, smooth animations, multiple difficulty levels, and classic gameplay mechanics. Includes nostalgic sound effects and pixel-perfect recreation of the original interface.

## Features

- **Authentic Recreation**: Pixel-perfect recreation of the original Windows 98 Minesweeper
- **Retro Styling**: Authentic Windows 98 visual design and styling
- **Multiple Difficulty Levels**: Easy, Medium, and Hard difficulty settings
- **Smooth Animations**: Fluid animations for tile reveals and flag placement
- **Sound Effects**: Nostalgic sound effects matching the original game
- **Classic Gameplay**: Faithful recreation of original game mechanics

## Game Mechanics

- **Mine Detection**: Classic mine detection and flagging system
- **Number Reveals**: Automatic number tile reveals when safe areas are found
- **Timer**: Classic game timer for tracking completion time
- **Mine Counter**: Real-time mine counter showing remaining mines
- **Win/Lose Conditions**: Proper win and lose state handling

## Technical Implementation

- **Phaser 3**: Professional HTML5 game development framework
- **HTML5 Canvas**: High-performance graphics rendering
- **JavaScript**: Modern ES6+ JavaScript features
- **Retro Styling**: Authentic Windows 98 CSS styling
- **Game State Management**: Robust game state and progress tracking
