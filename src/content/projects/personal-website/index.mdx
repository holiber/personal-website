# Personal Website

A modern, responsive personal portfolio website showcasing projects, skills, and professional experience. Built with Astro for optimal performance, TypeScript for type safety, and modern CSS for beautiful design. Features include project galleries, blog integration, and contact forms.

## Features

- **Modern Design**: Clean, professional design with modern aesthetics
- **Responsive Layout**: Optimized for all device sizes
- **Project Showcase**: Comprehensive project gallery with detailed descriptions
- **Blog Integration**: Integrated blog for sharing knowledge and updates
- **Contact Forms**: Interactive contact forms for easy communication
- **Performance Optimized**: Built with Astro for optimal loading speeds
