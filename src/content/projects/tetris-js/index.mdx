# Tetris JS

A classic Tetris implementation using the Cocos2d-js game framework. Features smooth gameplay mechanics, line clearing animations, score tracking, and responsive controls. Built with modern JavaScript practices and optimized for web browsers.

## Features

- **Classic Gameplay**: Faithful recreation of the classic Tetris experience
- **Smooth Animations**: Fluid line clearing and piece movement animations
- **Score Tracking**: Comprehensive scoring system with high score tracking
- **Responsive Controls**: Intuitive keyboard and touch controls
- **Web Optimization**: Optimized for web browser performance

## Technical Implementation

- **Cocos2d-js**: Professional game development framework
- **JavaScript**: Modern ES6+ JavaScript features
- **Canvas Rendering**: High-performance graphics rendering
- **Game Physics**: Realistic piece movement and collision detection
